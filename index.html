Serviços

  • Upgrade de componentes

    Fazemos a substituição de componentes, tanto para aumento de performance, quando para danificadas, claro dependendo da configuração do equipamento.

  • Montagem

    Faço diversos tipos de montagens de computadores desktops, domiciliar, escritório e pcgamers, tudo com base no orçamento do cliente.

  • Manutenção

    Fazemos diversos tipos de manutenção, qualquer componente danificado, peças deterioradas e qualquer substituição por tempo de uso.

  • Montagem de PC gamer

    A montagem de PCGAMER é muito aleatório e relativo, o cliente fala o que precisa e agente indica a melhor opção, até chegar a uma conclusão; Este equipamento não precisa ser uma super máquina, mas quanto maior recursos e performance, maior é o custo.

  • Formatação

    Fazemos em qualquer tipo de sistema operacional, WINDOWS, LINUX e MACOS, atualização de sistema e substituição, com tudo baseado na configuração.

  • Instalação de software etc ...

    Você tem dificuldade em instalar programas, em qualquer tipo versionamento, em qualquer tipo de sistema operacional? Entre em contato, podemos ajudar!

  • Criação de Website

    Quer criar um site bacana? Com efeitos e cores elegantes, atrair novos clientes, ficar gravado na cabeça das pessoas. Isto irá agregar valor a marca e passar uma imagem de empresa seria e importante, além de tirar dúvidas antes mesmo do interessado entrar em contato.

  • Manutenção de Web site

    Se você já tem um web site e quer mudar uma foto, um texto, cores, fontes ou qualquer outro assunto que queira modificar, nós também cuidamos disso para você.

  • Cartão de visita online

    Já imagino entregar um cartão de visita online?? Sim é possível, tanto quanto deixar disponível na sua bio, facebook e Instagram quanto em qualquer outro lugar.

  • Paginas estáticas

    Precisa de uma página para divulgar seu trabalho de forma simples? A página estática é para isso, são sites de página única sem efeitos e de valor econômico.

  • Landing Page

    É uma página estática mais atraente e com alguns efeitos, o seu principal objetivo, é capturar clientes e levar para o seu contato direto, para converter em vendas.

  • One Page

    Este tipo de site é um site de página única, com efeito móbile, é como se seu site se comporta-se da mesmo forma em uma tela de desktop, quanto no móbile.
